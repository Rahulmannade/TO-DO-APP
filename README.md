# TO-DO-APP
A simple and smart To-Do List web application built using HTML, CSS, and JavaScript. This app helps users manage daily tasks efficiently with features like task prioritization and deadline tracking.
# 📝 To-Do List / Task Manager

--

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 💾 Data stored using LocalStorage (no data loss on refresh)
* 🔥 Task priority (High / Medium / Low)
* ⏰ Deadline tracking

---

```

```

2. Open the folder
3. Double-click on `index.html` or open it in browser

## 🚀 Deployment (GitHub Pages)

1. Go to your repository
2. Click on **Settings**
3. Scroll to **Pages** section
4. Select branch: `main`
5. Save and get your live link
